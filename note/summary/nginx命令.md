1. ubuntu安装nginx

   ```
   sudo apt-get install nginx
   ```

2. 启动

   ```
   sudo nginx  或 serveice nginx start
   ```

3. 关闭

   ```
   sudo nginx -s stop
   ```

4. 查看配置文件是否正确

   ```
   nginx -t
   ```

5. 重新加载配置文件

   ```
   sudo nginx -s reload
   ```

6. 重新打开配置文件

   ```
   sudo nginx reopen
   ```
